Investigacion JS

¿Qué es el lenguaje JavaScript?

JavaScript es un lenguaje de programación interpretado, conocido por ser el lenguaje de la web. Se utiliza para crear páginas web interactivas.

¿Para qué sirve el lenguaje JS?

JavaScript se utiliza principalmente para el desarrollo web, permitiendo la creación de contenido dinámico y controlando multimedia, animaciones, etc.

¿A qué se refiere el DOM (Document Object Model) en el desarrollo Web?

El DOM es una interfaz de plataforma independiente que permite a los programas y scripts acceder y actualizar el contenido, estructura y estilo de los documentos HTML y XML.

¿Cuál es la diferencia entre los archivos HTML, CSS y JS?

HTML se utiliza para estructurar el contenido en la web, CSS se usa para diseñar y dar estilo a los elementos HTML, y JavaScript se emplea para agregar interactividad a las páginas web.

¿Para qué sirve la etiqueta <script>?

La etiqueta <script> se utiliza para insertar código JavaScript en un documento HTML.

Los 3 métodos para cambiar elementos tipo Tag Name, ID y Class Name con JS

Los tres métodos son: getElementById para ID, getElementsByClassName para clases y getElementsByTagName para nombres de etiqueta.

Programa básico "Hola mundo" en JS

Explicación: Este código JavaScript utiliza la función document.write() para escribir "Hola mundo" directamente en el documento HTML.

¿Cómo se declaran funciones en JS?

    return "Hola";
}

¿Cómo se llaman funciones en JS?

Investigacion PHP

¿Qué es el lenguaje PHP?

PHP es un lenguaje de programación del lado del servidor diseñado para el desarrollo web.

¿Para qué sirve el lenguaje PHP?

PHP se utiliza principalmente para crear páginas web dinámicas y para interactuar con bases de datos.

¿Cuál es la diferencia entre los archivos HTML y PHP?

HTML es un lenguaje de marcado utilizado para estructurar contenido, mientras que PHP es un lenguaje de programación que se ejecuta en el servidor para generar contenido dinámico en las páginas web.

¿Para qué sirve la etiqueta <?php ?>?

La etiqueta <?php ?> se utiliza para iniciar y cerrar un bloque de código PHP.

Programa básico "Hola mundo" en PHP

                            
<?php
    echo "Hola mundo";
?>

Explicación: Este código PHP utiliza la función echo para imprimir "Hola mundo" en la página web. La etiqueta <?php ?> indica el inicio y el fin del bloque de código PHP.

¿Cómo se declaran variables en PHP?

                            
$variable = "valor";

¿Cómo se declaran funciones en PHP?

                            
function saludo() {
    return "Hola";
}

¿Cómo se llaman funciones en PHP?

                            
echo saludo();

¿Cuáles ciclos de programación están disponibles en PHP?

                            
for ($i = 0; $i < 10; $i++) { ... }
while ($condicion) { ... }
do { ... } while ($condicion);
foreach ($array as $valor) { ... }

¿Cuáles funciones de condicional están disponibles en PHP?

                            
if ($condicion) { ... }
elseif ($condicion) { ... }
else { ... }
switch ($expresion) { ... }

Ejemplo de manejo de archivos en PHP

                            
<?php
    if (!file_exists('datos.txt')) {
        file_put_contents("datos.txt", "0\r\n");
    }

    if (isset($_GET['DATO'])) {
        $DATO_var = $_GET['DATO'];
        $TEXTO = $DATO_var . "\r\n";
        file_put_contents("datos.txt", $TEXTO);
    }

    $ARCHIVO = file_get_contents("datos.txt");
    $pos1 = strpos($ARCHIVO, "\r\n");
    $DATO_lectura = substr($ARCHIVO, 0, $pos1);
?>

<!DOCTYPE html>
<html>
<head>
    <title>Mi primer página web</title>
    <meta http-equiv="refresh" content="5">
</head>
<body>
    <h1>
        <?php
            echo $DATO_lectura;
        ?>
    </h1>
</body>
</html>

Explicación: Este código PHP crea y maneja un archivo de texto. Si el archivo no existe, lo crea. Luego, guarda el dato recibido por la URL en el archivo. Finalmente, lee y muestra el primer dato del archivo en la página web.

¿Cómo se utiliza el método Open/Read en el manejo de archivos con PHP?

                            
$file = fopen("archivo.txt", "r");
$contenido = fread($file, filesize("archivo.txt"));
fclose($file);

¿Cómo se utiliza el método Create/Write en el manejo de archivos con PHP?

                            
$file = fopen("archivo.txt", "w");
fwrite($file, "Contenido del archivo");
fclose($file);
